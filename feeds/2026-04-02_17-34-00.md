I was organizing some bathroom items when I found an item I thought I did not have and bought recently an item, so I have duplicated. Today and in other previous times, I have thought about some app to manage the inventory, so I could avoid these things.

Some time ago I thought about creating an inventory app to run as PWA and that can be used offline from mobile using some persistence mechanism in the browser.

Searching, I have found a self-hosted app that can be used from mobile too. It has no PWA feature or offline usage but I think it could be useful to me as starting point to create the inventory. This allows exporting to CSV so I could migrate the data.

The app is called [Homebox](https://homebox.software/en/) and it has the minimal requirements for me as nested storage location and other cool features.
